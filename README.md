# customer_churn
# 📊 Telecom Customer Churn Dashboard

An interactive Streamlit dashboard for analyzing telecom customer churn. The dashboard enables users to explore customer behavior, identify churn patterns, and gain insights using interactive visualizations and key performance indicators (KPIs).

---

## 🚀 Features

- Interactive dashboard built with Streamlit
- Sidebar filters for:
  - Area Code
  - International Plan
  - Voice Mail Plan
  - Churn Status
- KPI Cards
  - Total Customers
  - Churn Rate
  - Average Day Minutes
  - Average Customer Service Calls
- Interactive Charts
  - Customer Churn Distribution
  - Day Minutes vs Churn
  - Day vs Evening Minutes Scatter Plot
  - Customer Service Calls Distribution
  - International Plan Distribution
  - Voice Mail Plan Distribution
- Correlation Heatmap
- Searchable Data Table
- Download Filtered Dataset

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Telecom_Churn.git
```

Move into the project folder

```bash
cd Telecom_Churn
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run app.py
```
